================================================================================

   - citing_papers_analysis.py
  - Purpose: Forward snowballing (papers citing yours)

- scripts/snowball/backward/
  - fetch_references_backward.py
  - remove_excluded.py
  - keep_level_papers.py
  - fetch_info_refferences.py
  - fetch_info_refferences_level.py
  - references_analysis.py
  - Purpose: Backward snowballing (papers your set references)

WORKFLOW SEQUENCE (updated):
================================================================================

1. relevance_score.py
   -> Scores papers from Scopus CSV for relevance to HMLV/DSS

2. scored_csv_to_json.py
   -> Converts scored CSV to JSON with unique key_id identifiers

3. snowball/backward/fetch_references_backward.py
   -> Fetches references (backward snowballing) for scored papers

4. fetch_citing_papers.py
   -> Fetches papers citing your papers (forward snowballing)

5. snowball/backward/remove_excluded.py
   -> Filters excluded papers from backward reference chain

6. snowball/backward/keep_level_papers.py
   -> Filters papers by relevance level (L0, L3, L4)

7. snowball/backward/fetch_info_refferences.py
   -> Enriches backward reference data with metadata

8. snowball/forward/fetch_citing_papers_level.py
   -> Forward snowballing for selected level-filtered papers

9. snowball/forward/citing_papers_analysis.py
   -> Analyzes impact and distribution of citing papers

10. snowball/backward/references_analysis.py
    -> Analyzes statistics on referenced papers
